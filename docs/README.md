# GitHub + Figma Mixer — Project Documentation

**GitHub + Figma Mixer — Spring Merge: Where Code Meets Design** is a Next.js event landing page for a joint GitHub × Figma networking event held on March 19, 2026 at SPRING VALLEY BREWERY TOKYO. The page is written in TypeScript with React 19 and Tailwind CSS v4, statically exported, and deployed to GitHub Pages.
