# GitHub + Figma Mixer — Project Documentation

**GitHub + Figma Mixer — Spring Merge: Where Code Meets Design** is an event landing page for the GitHub × Figma networking mixer held on **2026年3月19日 (Thursday)** at **SPRING VALLEY BREWERY TOKYO**. The site is built with Next.js 16.1.7, React 19.2.3, Tailwind CSS v4, and TypeScript, and features a Copilot-powered chat assistant via `@github/copilot-sdk`.
